# Überblick

> Öffentliche institutionelle Dokumentation für professionelle Gegenparteien, Aufsichtsbehörden und Wirtschaftsprüfer.

Diese Dokumentation definiert den institutionellen Umfang, die architektonischen Grenzen, die Governance-Haltung und die Compliance-Positionierung von Aureum One.

Es richtet sich an professionelle Gegenparteien, Depotbanken, Ausführungsplätze, Wirtschaftsprüfer, Aufsichtsbehörden und institutionelle Partner.

Diese Dokumentation ist kein Marketingmaterial und stellt kein Angebot, keine Aufforderung oder Finanzberatung dar.
