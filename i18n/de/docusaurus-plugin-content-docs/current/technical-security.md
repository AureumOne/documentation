Die Sicherheitsprinzipien umfassen:

- Geringste Berechtigung und Rollentrennung,
- Obligatorische MFA für Administratoren,
- Unveränderliche Protokollierung über CloudTrail,
- Netzwerkisolation über VPC-Subnetze,
- Regelmäßige Sicherheitstests und Backups,
- Institutionelle Härtung von Infrastrukturkomponenten.
