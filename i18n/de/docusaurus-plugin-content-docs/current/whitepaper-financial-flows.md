## 5.1 Kaufablauf
1. Der Kunde reicht einen Kaufauftrag ein.
2. Mittel werden über institutionelle Konten empfangen.
3. Compliance-Prüfungen werden durchgeführt.
4. Aureum One erwirbt Gold oder ordnet aus dem Bestand zu.
5. Der Verwahrer bestätigt die Goldposition.
6. Die interne Zuordnung wird aktualisiert.
7. Token wird 1:1 auf XRPL geprägt.
8. Der Prüfer überprüft Stichproben und kritische Ereignisse.

## 5.2 Rücknahmeablauf
1. Der Kunde fordert die Rücknahme an.
2. Token wird verbrannt oder in Reserve verschoben.
3. Gold wird liquidiert oder der Bestand angepasst.
4. Fiat wird an den Kunden zurückgegeben.
5. Vollständiger Prüfpfad wird aufgezeichnet.
