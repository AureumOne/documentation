> Strukturelle Trennung, die sicherstellt, dass keine einzelne Einheit die Verwahrung, Ausführung, Governance und Beweise kontrolliert.

Aureum One unterteilt das System konstruktionsbedingt in **vier unabhängige Schichten**.

### Sorgerechtsebene
Physisches Gold wird von professionellen Drittverwahrern im Rahmen ihrer eigenen rechtlichen, regulatorischen und versicherungstechnischen Rahmenbedingungen verwahrt.

Aureum One **hat keinen Zugriff auf Tresore, Metall oder Depotkonten**.

### Ausführungsebene
Der Handel wird von unabhängigen Edelmetallhäusern oder Ausführungsplätzen auf ihren eigenen Plattformen ausgeführt.

Preis-, Liquiditäts- und Ausführungsrisiko **bleiben völlig außerhalb von Aureum One**.

### Governance-Schicht
Aureum One erzwingt **nicht diskretionäre Governance-Regeln**, einschließlich Genehmigungsanforderungen, Autorisierungsreihenfolge, Rollentrennung und Eskalationspfade.

### Beweisschicht
Jedes geregelte Ereignis generiert **unveränderliche, nur anhängbare Beweise**, die für externe Prüfungen und behördliche Überprüfungen bestimmt sind.
