> Por qué la evidencia se trata como un activo institucional, no como un registro operativo.

En Aureum One, **la evidencia no se registra**.

**La evidencia se trata como un activo institucional.**

Cada evento crítico genera un registro sellado con **integridad criptográfica, estructura determinista y compatibilidad con auditorías externas**.

La evidencia es **solo para agregar y no se puede modificar ni eliminar**.
