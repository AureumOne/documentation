title: Modelo de Seguridad
sidebar_label: Seguridad
---

Los principios de seguridad incluyen:

- Privilegio mínimo y segregación de roles,
- MFA obligatorio para administradores,
- Registro inmutable vía CloudTrail,
- Aislamiento de red vía subredes VPC,
- Pruebas de seguridad y respaldos regulares,
- Endurecimiento institucional de componentes de infraestructura.
