# Descripción general

> Documentación institucional pública para contrapartes profesionales, reguladores y auditores.

Esta documentación define el alcance institucional, los límites arquitectónicos, la postura de gobernanza y el posicionamiento de cumplimiento de Aureum One.

Está destinado a contrapartes profesionales, custodios, centros de ejecución, auditores, reguladores y socios institucionales.

Esta documentación no es material de marketing y no constituye una oferta, solicitud o asesoramiento financiero.
