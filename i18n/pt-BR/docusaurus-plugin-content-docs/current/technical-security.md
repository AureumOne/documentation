sidebar_label: Segurança

Os princípios de segurança incluem:

- Privilégio mínimo e segregação de funções,
- MFA obrigatório para administradores,
- Logging imutável via CloudTrail,
- Isolamento de rede via sub-redes VPC,
- Testes de segurança e backups regulares,
- Hardening institucional de componentes de infraestrutura.
