## Privacidade e proteção de dados

> Como Aureum One processa, protege e administra dados institucionais e pessoais.

Aureum One processa dados institucionais de acordo com os regulamentos de proteção de dados aplicáveis.

Os dados pessoais são processados ​​apenas na medida necessária para:
- aplicação de regras de governança
- fluxos de trabalho de autorização
- geração de evidências
- conformidade regulatória

Aureum One não vende, compartilha ou monetiza dados do usuário.

Os dados são armazenados de forma segura e o acesso é restrito a pessoal e sistemas autorizados.

Os clientes mantêm a propriedade dos seus dados e podem solicitar acesso, correção ou exclusão de acordo com a legislação aplicável.

As contrapartes terceirizadas (custodiantes, locais de execução) mantêm suas próprias políticas de privacidade e práticas de processamento de dados.

Aureum One não é responsável por atividades de processamento de dados de terceiros.

Para dúvidas sobre proteção de dados, entre em contato com Aureum One através dos canais oficiais.
