## 5.1 Fluxo de Compra
1. O cliente submete uma ordem de compra.
2. Os fundos são recebidos através de contas institucionais.
3. Verificações de compliance são realizadas.
4. Aureum One adquire ouro ou aloca do inventário.
5. O custodiante confirma a posição de ouro.
6. A alocação interna é atualizada.
7. O token é mintado 1:1 no XRPL.
8. O auditor revisa amostras e eventos críticos.

## 5.2 Fluxo de Resgate
1. O cliente solicita o resgate.
2. O token é queimado ou movido para reserva.
3. O ouro é liquidado ou o inventário é ajustado.
4. O fiat é devolvido ao cliente.
5. A trilha de auditoria completa é registrada.
