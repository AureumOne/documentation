# Visão geral

> Documentação institucional pública para contrapartes profissionais, reguladores e auditores.

Esta documentação define o escopo institucional, limites arquitetônicos, postura de governança e posicionamento de conformidade de Aureum One.

Destina-se a contrapartes profissionais, custodiantes, locais de execução, auditores, reguladores e parceiros institucionais.

Esta documentação não é material de marketing e não constitui uma oferta, solicitação ou aconselhamento financeiro.
